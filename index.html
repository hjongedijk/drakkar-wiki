Build a media library that actually stays organized.

Drakkar syncs Seerr requests, searches with NZBHydra2, streams direct and archived releases through FUSE, and keeps Plex-facing library paths stable without turning your storage into a mess.

Direct streaming

Serve media before full extraction.

Direct files and archive-backed releases can become streamable through the VFS layer, with targeted Plex refresh instead of broad library churn.

Request sync

Seerr requests move fast.

Polling still works, but webhooks can push approved requests straight into Drakkar so they get prioritized near the front of the queue without interrupting an active download. Periodic bulk request sync is now slower by default and mainly handles reconciliation for larger libraries.

Naming control

Canonical movie and TV paths.

Import, relink, and cleanup flows keep a single canonical library path per title so Plex and your storage do not drift into duplicate show folders.

    Bazarr-compatible Arr shims that accept the same Drakkar API token.
