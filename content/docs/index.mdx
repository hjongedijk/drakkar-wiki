# Drakkar Docs

Drakkar is a Usenet-first media automation stack with:

- request sync from Seerr
- NZBHydra2-backed search and feed caching
- direct and archived streaming through FUSE
- Plex-targeted refreshes
- a library view that tracks requested, downloading, missing, and completed media
- GitHub-backed documentation pages editable through normal repository files
