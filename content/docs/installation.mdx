# Installation

## Public Compose

Use the published compose stack from the backend release repo.

Services included:

- `drakkar_frontend` (port 8080)
- `drakkar_backend` (port 3000, internal)
- `drakkar_rclone` (FUSE mount at `/mnt/drakkar/vfs`)
- `drakkar_postgres` (internal)
- `drakkar_valkey` (internal)
- `drakkar_valkey_insight` (port 5540)
- `drakkar_nzbhydra2` (port 5076)
- `drakkar_apprise_api` (port 8000)
- `drakkar_seerr` (port 5055)

Start:

```bash
mkdir -p ./data
docker compose up -d
```

Open `http://<server-ip>:8080`.

## Host Mount Setup

Drakkar mounts the FUSE VFS at `/mnt/drakkar/vfs` and stores media symlinks at `/mnt/drakkar/media`. The backend, rclone, and Plex must all see the same `/mnt/drakkar` tree.

### Systemd unit

Create `/etc/systemd/system/drakkar-mount.service`:

```ini
[Unit]
Description=Prepare Drakkar shared bind mount
After=local-fs.target
Before=docker.service

[Service]
Type=oneshot
ExecStartPre=/usr/bin/mkdir -p /mnt/drakkar
ExecStartPre=/usr/bin/mkdir -p /mnt/drakkar/media/movies
ExecStartPre=/usr/bin/mkdir -p /mnt/drakkar/media/tv
ExecStartPre=/usr/bin/mkdir -p /mnt/drakkar/vfs
ExecStartPre=/usr/bin/mkdir -p /mnt/.drakkar-vfs-store
ExecStartPre=/usr/bin/sh -c 'printf "%s\n" "vm.overcommit_memory = 1" > /etc/sysctl.d/99-valkey.conf'
ExecStartPre=/usr/sbin/sysctl -p /etc/sysctl.d/99-valkey.conf
ExecStart=/usr/bin/mount --bind /mnt/drakkar /mnt/drakkar
ExecStart=/usr/bin/mount --make-rshared /mnt/drakkar
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target
```

Enable:

```bash
sudo systemctl daemon-reload
sudo systemctl enable --now drakkar-mount.service
findmnt -T /mnt/drakkar -o TARGET,PROPAGATION
```

Expected propagation: `shared` or `rshared`.

### Plex container bind

```yaml
volumes:
  - ./plex-data/config:/config
  - /mnt/drakkar:/mnt/drakkar:rslave,z
```

Configure Plex libraries to:

- Movies: `/mnt/drakkar/media/movies`
- TV Shows: `/mnt/drakkar/media/tv`

## First Boot

Open the frontend and complete the setup wizard:

1. Create the first admin account
2. Connect NZBHydra2
3. Connect a Usenet provider
4. Connect Seerr
5. Connect metadata providers (TMDB, TVDB)
6. Connect Plex

## Recommended Request Flow

After setup:

- configure the Seerr webhook: `http://YOUR_HOST:8080/api/webhooks/seerr`
- keep periodic request sync enabled (runs every 10 minutes)

The webhook handles immediate single-request delivery; the scheduled sync handles reconciliation and recovery.

## Companion Service Startup

- `drakkar_backend` waits for `drakkar_postgres` and `drakkar_valkey`
- `drakkar_rclone` waits for `drakkar_backend` (health check)
- `drakkar_frontend` waits for `drakkar_backend`
- `drakkar_valkey_insight` waits for `drakkar_valkey`
- `drakkar_nzbhydra2` waits for `drakkar_apprise_api`
- `drakkar_seerr` and `drakkar_apprise_api` start independently
